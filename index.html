SFR

Développement d'une plateforme pour les applications mobile du groupe Altice. La plateforme a pour objectif de permettre :

  • installation et mise à jour des applications sur les téléphones SFR
  • push vers les applications
  • API REST de gestion des applications (vérifier les mises à jour, remontées statistiques,…)
  • certains services spécifiques à des applications
  • l'exploitation de toutes les statistiques à des fins de support et marketing
Mon rôle dans ce projet est de développement, l'architecture, la maintenance opérationnelle , les tests et les benchmarks.
